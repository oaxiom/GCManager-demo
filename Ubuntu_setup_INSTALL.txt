Instructions to set up Ubuntu:

1. Install Ubuntu 24.04 LTS
2. Add a new user gcm, password of your choice.
3. Download/copy the 'tools.tar.gz' package. Unpack in ~
4. Download/copy the 'static_data.tar.gz' package. Unpack in ~
5. (Optional) Download the test_data.tar.gz package. Unpack in ~
6. Setup the python venv:
    a. $ sudo apt install python3-venv
    b. $ python3 -m venv ~/gcmvenv
    c. $ source ~/gcmvenv/activate
    d. Add 'source ~/gcmvenv/activate' to ~/.profile and ~/.bashrc (BUG?)
7. Install slurm in one node configuration:
    a. sudo apt install slurmd slurmctld -y
    b. sudo chmod 777 /etc/slurm/
